Lesson 4: Introduction to GitHub

What is GitHub?

  • GitHub is a popular online platform where you can store, manage, and collaborate on Git repositories. It acts like a bridge between developers, enabling them to share their code, track changes, and work together efficiently on projects.

  • Think of GitHub as a digital workspace for developers. It provides tools to organize and maintain code projects, whether you're working solo or with a team.

Key Steps to Use GitHub

  • 1. Create a GitHub Account: Visit GitHub and sign up for a free account. It's the first step to getting started with GitHub.

  • 2. Create a New Repository: A repository is where your project lives on GitHub. To create one:

    1. Log in to your GitHub account.
    2. Click the green "New" button in the repositories section.
    3. Provide a name for your repository (e.g., "MyFirstRepo").
    4. Choose visibility (Public or Private) and click "Create Repository."

  • 3. Link Your Local Git Repository to GitHub: Use the command below in your terminal to link your local repository with the GitHub repository: 

    git remote add origin https://github.com/username/repository.git
    Replace username with your GitHub username and repository with your repository name.

  • 4. Push Your Changes to GitHub: Once your work is ready, push it to GitHub with the following command: 

    git push -u origin main
    This uploads your changes to the main branch of the GitHub repository.

Practical Exercise

  • Step 1: Create a new repository on GitHub.

  • Step 2: Initialize a local Git repository, add some code, and commit your changes.

  • Step 3: Link the local repository to your GitHub repository using the git remote add command.

  • Step 4: Push your changes to GitHub using git push.

  • Step 5: Open GitHub and view your uploaded project.

Once you complete these steps, your code will be safely stored online, accessible from anywhere, and ready for collaboration.
